Layout breakdown

  • Sidebar (220px) — Agent list only. No KB filter. Selected agent highlighted with accent border.
  • Context bar — Shows selected agent name + status + actions.
  • Config panel (380px, scrollable) — All structured sections: identity, style, guardrails, behavior, retriever tools, additional instructions. Save button at bottom.
  • Playground (flex, remaining space) — Single chat session with this agent. Shows tool calls as compact expandable chips above the response. Streaming cursor on active response. Clear chat to reset.
  • Retriever tools — Listed with their source KB name + config details. "+ Add retriever from any KB" to pick from all org retrievers.
